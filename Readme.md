## 요구사항
- 주어진 횟수동안 n대의 자동차는 전진 또는 멈출수있다.
- 사용자는 몇 번의 이동을할 것인지를 입력 할 수 있어야한다.
- 전진하는 조건은 0에서9사이에서 random값을 구한 후 random값이 4이상일 경우 전진하고, 3이하의 값이면 멈춘다.

---
- 0~9 사이의 랜덤값을 구한다
  - 랜덤값이 4이상이면 전진을 리턴한다
  - 랜덤값이 3이하면 멈춤을 리턴한다
